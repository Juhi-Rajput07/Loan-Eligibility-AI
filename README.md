# 🏦 Loan Eligibility & Fraud Detection System

An End-to-End Machine Learning application that predicts loan approval status and identifies high-risk fraudulent applications.

## 🚀 Features
- Random Forest Classifier: Trained on 50,000+ records with 99% accuracy.
- Automated Feature Engineering: Handles CIBIL scores, Debt-to-Income ratios, and transaction history.
- Interactive UI: Built with Streamlit for real-time risk assessment.

## 🛠️ Tech Stack
- Python (Pandas, Scikit-Learn)
- Streamlit (Web Interface)
- Joblib (Model Serialization)
